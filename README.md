# Thomas Sonzogni — Portfolio

Site personnel professionnel présentant mon parcours, mes compétences et mes projets en tant que Data Analyst.

## Aperçu

Site one-page responsive en noir et blanc, conçu pour être consulté aussi bien sur PC que sur mobile. Il regroupe l'ensemble de mon profil professionnel : storytelling, projet phare, expériences, formation et compétences techniques.

## Sections

- **Accueil** — Présentation rapide, photo et disponibilité
- **À propos** — Mon parcours et ce qui me motive
- **Projet phare** — OPAGE, outil de pilotage des dommages réseau chez Orange (Power Apps, SharePoint, Power BI)
- **Expériences** — Timeline interactive : Orange, Australie, Pompier professionnel, BNP Paribas, bénévolat
- **Formation** — Data Analyst (OpenClassrooms / CFA Orange) et Licence Économie et Gestion
- **Compétences** — Techniques, méthodologies et soft skills
- **Inspiration** — Citation de Sénèque
- **Contact** — Email, téléphone, LinkedIn

## Stack technique du site

HTML · CSS · JavaScript — aucune dépendance externe hormis Google Fonts.

## Fonctionnalités

- Navigation fixe avec masquage automatique au scroll
- Menu burger animé sur mobile
- Barre de progression de lecture
- Animations d'apparition au scroll (Intersection Observer)
- Expériences en accordéon (clic pour déplier)
- Design responsive (mobile, tablette, desktop)

## Déploiement

Le site est hébergé via GitHub Pages. Toute modification poussée sur la branche principale est automatiquement déployée.
